# Лабораторный блок питания

Линейный лабораторный блок питания, спроектированный с ориентацией
на классическую инженерную школу (HP / Siemens).

## Цели проекта
- Высокая надёжность
- Предсказуемые тепловые режимы
- Низкий уровень шума
- Понятная и документированная архитектура
- Отсутствие экзотических и плохо воспроизводимых решений

## Основные особенности
- Однополярный режим с повышенным током нагрузки
- Двухполярный ± режим для аналоговых схем
- Силовая часть на сетевых трансформаторах
- Линейная стабилизация напряжения и тока
- Архитектурный подход вместо копирования готовых схем

## Состояние проекта
Формирование архитектуры и требований.
